# LEONARDO JAQUES // ARCHITECT_CORE_v2.0

Portfolio pessoal v2 - Site estatico com estetica terminal/cyberpunk.

## Funcionalidades

- **i18n PT/EN** - Toggle de idioma na navbar e no modal de privacidade
- **LGPD Cookie Consent** - Modal de consentimento com persistencia via cookie
- **Umami Analytics** - Carregado condicionalmente apos aceitacao de cookies
- **Scroll Animations** - Fade-up com IntersectionObserver
- **Smooth Scroll** - Navegacao suave entre secoes
- **Filtro de Projetos** - Todos / Web / Sistemas
- **Timeline de Carreira** - Historico profissional com linha do tempo
- **Responsivo** - Layout adaptado para mobile com bottom navbar
- **Efeito Scanline** - No hero section

## Secoes

1. **Hero** - Apresentacao com imagem do Cristo Redentor
2. **Core Competencies** - Solution Architect, Tech Lead, Software Engineer
3. **Projetos** - 6 projetos com filtro por categoria
4. **Onde Trabalhei** - Timeline: Mutant/Vivo, L.J-tech, Redspark, To-brasil, BRQ
5. **Tecnologias** - 6 categorias: Linguagens, Frameworks, AI/LLMs, Cloud/DevOps, Dados, Arquitetura
6. **Sobre** - Bio, stats (8+ anos, 229+ repos)
7. **Contato** - Email, WhatsApp, LinkedIn

## Tech Stack

- HTML5 / CSS3 / JavaScript
- Tailwind CSS (via CDN)
- Google Fonts (Space Grotesk, Inter, Fira Code)
- Material Symbols (icons)

## Como rodar

```bash
cd site-v2
python3 -m http.server 8080
```

Acesse `http://localhost:8080`
